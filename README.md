#Real-time Collaboration Application in Three.js
========
[Running Demo](http://storage.googleapis.com/hecodes/app/index.html)

Accompanying blog post [here](http://hecodes.com/?p=386&preview=true)

## Installation
Clone repository and run

```
npm install
```

## Usage
Run server
```
node app/server.js
```

Run client

```
npm run server
```
